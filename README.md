# Conversor-de-moneda
Conversor de monedas en Java
#Características 
Permite el cambio De moneda entre varios tipos de divisas
Permite hacer el cambio de moneda entre: 
Dólar 
Euros 
Libras Esterlinas 
Yen Japonés 
Won surcoreano 
Pesos Mexicanos
